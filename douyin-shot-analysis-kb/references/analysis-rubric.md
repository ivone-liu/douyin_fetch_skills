# Analysis rubric

## Evidence hierarchy

Strongest:
1. direct video observation
2. frame or subtitle sampling
3. normalized metadata with engagement metrics
4. comments as secondary signal

## Required dimensions

Every creator-level analysis should cover:
- hook design
- shot language
- editing rhythm
- narrative template
- packaging
- commercial intent

## Rule for conclusions

Do not write claims that cannot be traced back to one of these:
- repeated pattern across multiple videos
- outlier post with clear evidence
- metric association visible in the dataset

## Rule for usefulness

Each conclusion must end in one of these forms:
- do this
- test this
- avoid this
- use only when these conditions are true
