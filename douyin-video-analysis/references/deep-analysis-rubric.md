# Deep analysis rubric

A strong per-video analysis should answer these questions with evidence:

1. Was the local video really downloaded and probed?
2. What hard media facts do we know?
   - duration
   - resolution
   - orientation
   - fps
   - has audio
3. What editing facts can be estimated?
   - scene change count
   - avg shot length estimate
   - pace classification
4. What visual evidence files exist?
   - contact sheet
   - keyframes
5. What is still unknown?
   - transcript
   - exact shot types
   - semantic meaning of each frame

Never collapse unknowns into fake certainty.
